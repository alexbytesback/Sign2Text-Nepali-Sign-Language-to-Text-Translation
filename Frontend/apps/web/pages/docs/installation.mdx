# Installation

Sign2Text integrates seamlessly with your React-based project.

<Steps>
### Install

To add Sign2Text components to your site, you need to install **Next.js** (optional), **React**, and Sign2Text. In your project directory, run the following command to install Sign2Text and the dependencies:

```sh npm2yarn copy
npm i -D sign2text
```

### Initialize the Project

```sh copy
sign2text init
```

### Ready to Go!

Now, you can add Sign2Textcomponents to your project.

```sh copy
sign2text add button
```

</Steps>

That is it, all it takes to setup your **Sign2Text** Project
